## ğŸ“¦ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Muhammadasharmian/PaddleCoach.git
   cd PaddleCoach
   ```

2. **Open the project:**
   - Simply open `index.html` in your web browser
   - Or use a local server:
